## Getting started

### Build dependencies

* `make`
* `g++`

### `make` for dummies

1. Clone the repo

   ```bash
   git clone https://github.com/remicmacs/nutsh.git
   ```

2. Get in the clone repo

    ```bash
    cd nutsh
    ```

3. Build the shell with `make`
4. *Voilà* !!
